Topics for the final

The final exam for this course will be comprehensive, and will cover all of the topics we have already covered on the midterm exams along with new topics since midterm two.

Here is a list of topics we have covered since the second midterm:

  1. Parallel algorithms: know how to use the spawn, sync, and parallel for pseudocode keywords.
  2. String matching algorithms: know how the Rabin-Karp and Finite State string matching algorithms work.
  3. Computational graphics: know how the cross product operation works and how to use it to find intersections of line segments. Know how to compute a polar angle and how to use polar angles to implement Graham Scan and Jarvis's March.
  4. NP-completeness: know the definitions of key NP-complete problems such as formula satisfiability, the clique problem, vertex covers, Hamiltonial cycles, and the subset-sum problem.
  5. Approximation algorithms: be familiar with the approximation algorithms for vertex cover, the traveling salesman problem, and subset sum.

The exam will be more heavily weighted toward new topics. Here also is a priority list of important topics from previous exams that I may ask about:

  1. Using the master theorem to solve recurrence relations and estimate runtimes.
  2. Key graph algorithms, including BFS, DFS, Kruskal's and Prim's algorithms, and Dijkstra's algorithm.
  3. Using recursion to solve problems involving finding optimal subsets, such as the Knapsack problem or the subset sum problem.
