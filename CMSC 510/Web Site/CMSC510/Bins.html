The bin packing problem

Start by reading problem 35-1 in the textbook. This problem describes the bin-packing problem and gives one suggestion for getting an approximately optimal solution, the first-fit heuristic.

Here is another, less effective algorithm for doing bin packing:

  1. Start with one open bin that has no items in it.
  2. Scan across the items from left to right.
  3. Each time you come to a new item, try to put it in the current open bin. If it fits, put it in the bin.
  4. If the next item does not fit in the current open bin, close that bin and open a new, empty bin to put the item in.

What to do

Write a program that generates a random list of 12 items to be packed. The items should have integer weights randomly selected from the range 1-12. The bins have a maximum capacity of 20.

Start by trying to pack the items using the crude algorithm outlined above. Have your program print a count of how many bins were needed and print a listing of the items in each bin.

Next, try packing the items again using the first-fit heuristic described in problem 35-1. Print the number of bins required and what items are in each bin.

Finally, try using the crude algorithm in combination with some logic that tries every possible permutation of the 12 items. Have your program print the best packing it can find by that method. 12! is approximately 480 million, so give your program enough time to cycle through all the permutations.

Generating permutations

Here is some code to demonstrate how to generate and print all possible permutations of list of N items.

void swap(std::vector<int>& A, int i, int j) {
  if (i != j) {
    int temp = A[i];
    A[i] = A[j];
    A[j] = temp;
  }
}

void permute(std::vector<int>& A, int k) {
  if (k == N - 1)
    print(A);
  else {
    for (int j = k; j < N; j++) {
      swap(A, j, k);
      permute(A, k + 1);
      swap(A, j, k);
    }
  }
}

void printPermutations(std::vector<int>& A) {
  permute(A,0);
}
