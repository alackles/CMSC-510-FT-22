The problem

Write a C++ program to solve problem 15-4 from the text.

What to do

Construct your solution to 15-4 so that it reads all of the text from a file named "input.txt", prompts the user for a maximum line length, and then prints the text out to the console optimally broken into lines of length less than or equal to the given maximum.

You should use the text in this file to test your program.
