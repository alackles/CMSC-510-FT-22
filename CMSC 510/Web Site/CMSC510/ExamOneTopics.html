Midterm One Topics

Midterm one will cover chapters 2, 3, 4, 5, and 7. Here is a list of topics for you to review.

  • Estimating the best case and worst case run time for iterative algorithms.
  • Solving recurrences via a call tree, substitution, and the Master Theorem
  • Using random variables in run time estimates
  • Selection sort, insertion sort, merge sort, and Quicksort
  • Using loop invariants to prove that iterative algorithms are correct.
  • Using proof by induction to prove that recursive algorithms are correct.

The exam will be a closed book exam. I will allow you to bring a single sheet of paper with useful formulas on it to use as a reference.
